  • Scatter Plot es una gráfico de dispersion donde se puede observar la dispersión de los inputs entre 2 variables.
  • Bar Char es el gráfico de barras, donde se toman dos dimensiones para encerrar o mostrar un promedio de los datos.
  • El histograma es una variante del gráfico de barras que solo toma 1 columna de datos y se utiliza para contabilizar las frecuencias de los datos en un determinado rango.
  • Gráfica de pastel (pie chart) es una grafica para los datos relativos y compararlos

Estimación

  • Un problema de estimación es: dado un conjunto de datos, obtener una probabilidad o en dado caso, teniendo una probablidad, obtener los datos en base a esa probabilidad.

Average (Promedio)

  • Median(Mediana). Ordenados los datos, toma el valor que se encuentra justamente en el medio.
  • Mode(Moda). El dato más repetido en el data set.
